一个很好关于MATLAB效率的例子

显示预分配空间对速度的影响 显示矢量化对速度的影响 2016.11.1 aleko

Contents

  • 准备环境
  • 矢量化、预分配空间
  • 对照非矢量化、非预分配空间

准备环境

clear
num=1000;
num2=num*num;

矢量化、预分配空间

tic
A=zeros(num);
b=1:num2;
A=reshape(b,num,num);
toc

时间已过 0.002113 秒。

对照非矢量化、非预分配空间

tic
% C=zeros(num);% if missing, 1000 times slower in some occasions
for i=1:num
    for j=1:num
        C(j,i)=(i-1)*num+j;
    end
end
toc

时间已过 1.767909 秒。
